## 🧠 Miniguia de Estudo

### 📌 Resumo Estruturado

Naval Ravikant define riqueza como a posse de ativos que geram valor automaticamente.

Principais pilares:

* Riqueza ≠ dinheiro ≠ status
* Liberdade é o objetivo final
* Não se fica rico vendendo tempo
* Equity é essencial
* Alavancagem (código, mídia, capital) é o multiplicador
* Juros compostos geram crescimento exponencial

---

### 📖 Glossário

* **Riqueza:** ativos que geram renda passiva
* **Alavancagem:** uso de ferramentas para escalar resultados
* **Conhecimento específico:** habilidade única difícil de substituir
* **Equity:** participação em negócios
* **Accountability:** assumir responsabilidade pública
* **Juros compostos:** crescimento ao longo do tempo
